# Contributing Guidelines

We welcome contributions, but only small, clear, and safe changes.

## Rules
- One PR = one change
- No hardcoded AWS credentials
- Scripts must be free-tier safe
- Add a short explanation of WHAT and WHY
- Copy-paste from blogs without modification will be closed

## PRs will be closed if:
- No description is provided
- Changes are too large or vague
- Security or cost risks are ignored

## Supported tools
- AWS CLI
- Bash or Python only
